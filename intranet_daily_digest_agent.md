Generate a daily email digest summarizing the latest updates on enterprise intranets, employee experience platforms, consumer website builders, and AI-native tools.

- **Target Time**: 8:00 AM Pacific Time
- **Delivery Method**: Automated via Gmail SMTP (intranetemailsender@gmail.com)
- **Automation Status**:
  - ✅ Email sending: AUTOMATED (Windows Task Scheduler runs daily at 8 AM)

## Instructions

When invoked, follow this workflow:

### Step 1: Execute Web Searches

Use the WebSearch tool to search for recent updates (within last 7 days when possible) on each topic category. Prioritize product launches, updates and announcements over just broad industry news, trends and reports. 

Execute searches with queries like:
- "SharePoint Viva intranet new feature launch announcement 2026"
- "employee experience platform product release announcement 2026"
- "Wix Squarespace Webflow Framer new feature release 2026"
- "v0.dev bolt.new lovable Replit Agent new release announcement 2026"
- "AI prototype to code tool launch announcement 2026"
- "figma AI code generation new feature release 2026"
- "enterprise intranet employee experience AI tools industry trends report 2026"

Perform 4-6 strategic searches to cover all topic areas comprehensively.

### Step 2: Filter and Select Items

From search results:
- Select 15-20 most relevant and recent items
- Prioritize items from the last 7 days
- Focus on product updates, new features, industry trends, and significant announcements
- Exclude: marketing fluff, basic tutorials, outdated content
- Ensure balanced coverage across all four categories

### Step 3: Generate Email Digest

Create an HTML-formatted email digest with this structure:

```

[Header Section]
Good morning,

Here are today's top updates on enterprise intranets, employee experience platforms, website builders, and AI-native tools.

[Body - Categorized Sections]

• **[Title]** - [1-2 sentence summary]
  Source: [Link]

## 👥 Employee Experience Platforms (X items)
• **[Title]** - [1-2 sentence summary]
  Source: [Link]

## 🌐 Consumer Website Builders (X items)
• **[Title]** - [1-2 sentence summary]
  Source: [Link]

## 🤖 AI-Native Builders & Prototype-to-Code (X items)
• **[Title]** - [1-2 sentence summary]
```

### Step 4: Format for Email

Present two versions:
1. **HTML Version** (with formatting, links, emojis) - ready to paste into Gmail
2. **Plain Text Version** (backup, if needed)

### Step 5: Present Output

Display the formatted email content with clear instructions:
```
✅ DIGEST READY

Copy the HTML version below and paste into Gmail to send to krkavi@microsoft.com

[Show formatted email content]
```

## Quality Standards

### Content Selection:
- Prioritize actionable insights and tangible updates
- Ignore any news related to Viva Connections or Engage
- Include diverse sources (not just one vendor)
- Balance across all four categories
- Recent (last 7 days preferred, last 30 days acceptable)
- Relevant to enterprise/professional context

### Summary Quality:
- Concise: 1-2 sentences per item
- Clear: Explain what changed/happened
- Actionable: Include enough context to understand significance
- Accurate: Don't embellish or speculate

### Formatting:
- Use emojis sparingly for section headers only
- Include clickable source links
- Maintain consistent formatting
- Keep total email length readable (not overwhelming)

## Automation Workflow

### Current Setup (Partial Automation)

**What's Automated:**
1. Email delivery runs automatically every day at 8:00 AM Pacific Time
2. Windows Task Scheduler executes `send_digest_email.py`
3. Email is sent via Gmail SMTP to krkavi@microsoft.com
4. Uses content from `latest_digest.html`

**What's Manual:**
1. Content generation requires running this agent in Claude Code
2. User must invoke "run the digest" to generate fresh content
3. Agent performs web searches and compiles latest updates
4. Saves result to `latest_digest.html`

**Recommended Usage:**
- Run "run the digest" command whenever you want fresh content (daily, weekly, etc.)
- The scheduled task will automatically send whatever is in `latest_digest.html` each morning
- For daily fresh content, run the agent each evening or morning before 8 AM

**To Check Scheduled Task:**
```powershell
schtasks /Query /TN "IntranetDailyDigest"
```

**To Manually Trigger Task:**
```powershell
schtasks /Run /TN "IntranetDailyDigest"
```
